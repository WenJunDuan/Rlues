经验管理 — 操作统一的 `.ai_state/knowledge.md`:
- `vibe-exp show` — 显示 knowledge.md 全部段落摘要
- `vibe-exp add pitfall {描述}` — 追加到 "已知陷阱" 段
- `vibe-exp add pattern {描述}` — 追加到 "经验模式" 段
- `vibe-exp add decision {描述}` — 追加到 "架构决策" 段
- `vibe-exp add lesson {描述}` — 追加到 "教训" 段
- `vibe-exp search {关键词}` — 搜索 knowledge.md
