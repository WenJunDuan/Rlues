# Service Analysis Skill

## 用途

分析代码库的服务边界、依赖关系和集成点。
用于 vibe-plan 和 vibe-review 的架构分析。

## 分析内容

1. 服务/模块边界识别
2. 依赖关系图
3. API 接口清单
4. 数据流映射
5. 耦合度评估

## 输出

写入 `.ai_state/plan.md` 的架构分析部分。
