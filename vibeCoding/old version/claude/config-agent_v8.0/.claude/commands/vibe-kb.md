# vibe-kb

知识库操作。

## 语法

```bash
vibe-kb list               # 列出知识条目
vibe-kb search "关键词"    # 搜索知识库
vibe-kb add --category=tech "知识内容"  # 添加
```

## 知识库结构

```
.knowledge/
├── index.md          # 索引
├── project/          # 项目文档
├── standards/        # 开发规范
├── company/          # 团队约定
└── tech/             # 技术栈参考
```
