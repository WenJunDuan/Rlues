# Claude Code Official Commands Reference (v2.1+)

## Core Commands

| Command | Description |
|:---|:---|
| `/init` | Initialize project |
| `/plan` | Create implementation plan |
| `/todos` | Manage task list |
| `/review` | Code review |
| `/compact` | Compact context |
| `/model` | Switch model |
| `/config` | Configuration |
| `/help` | Show help |
| `/clear` | Clear conversation |
| `/exit` | Exit session |

## v2.1+ New Commands

| Command | Description |
|:---|:---|
| `/teleport` | Teleport session to web |
| `/rewind` | Rewind to checkpoint |
| `/tasks` | Background tasks |
| `/context` | Context management |
| `/mcp` | MCP server management |
| `/plugins` | Plugin management |
| `/theme` | Theme settings |

## Keyboard Shortcuts

| Shortcut | Action |
|:---|:---|
| `Ctrl+C` | Cancel current |
| `Ctrl+S` | Stats screenshot |
| `Ctrl+R` | Search history |
| `Ctrl+O` | Transcript mode |
| `Ctrl+B` | Background task |
| `Shift+Tab` | Auto-accept edits |
| `Esc Esc` | Rewind checkpoint |
