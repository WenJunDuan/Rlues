# Kaizen — 经验沉淀

## 触发: V 阶段

## 流程
1. 回顾本次开发
2. 提取模式: 什么有效? 踩了什么坑? 哪个插件组合好用?
3. → knowledge.md
4. 如有通用模式 → ECC instinct-export 导出

## knowledge.md 格式
```markdown
## [日期] [需求]
### 有效模式
- GSD execute 比 codex:rescue 更稳定 (因为 fresh context)
### 教训
- codex:adversarial-review 发现了 code-review 没发现的竞态
### 插件笔记
- Path B 用 GSD plan-phase + superpowers plan-review 组合最好
```
