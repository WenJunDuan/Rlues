# vibe-todos

## 类型
🔷 **增强官方指令** - 基于 `/todos`

## 用途
显示增强的任务列表，包含 Kanban 视图和进度追踪

## 语法
```bash
vibe-todos                    # 显示所有
vibe-todos --phase=<阶段>     # 按阶段过滤
vibe-todos --kanban           # Kanban 视图
vibe-todos --progress         # 进度统计
```

## 执行流程

```
vibe-todos
    │
    ▼
┌─────────────────────────────┐
│ 1. 调用官方 /todos          │ ← 官方 TODO 列表
│    - 显示标准 TODO          │
│    - 官方统计正常记录        │
└─────────────┬───────────────┘
              │
              ▼
┌─────────────────────────────┐
│ 2. 增强显示                 │
│    - Kanban 视图            │
│    - 进度百分比             │
│    - 关联文档               │
│    - 时间估算               │
└─────────────────────────────┘
```

## 官方 /todos vs vibe-todos

| 维度 | 官方 /todos | vibe-todos |
|:---|:---|:---|
| TODO 列表 | ✅ | ✅ 继承 |
| Kanban 视图 | ❌ | ✅ 增强 |
| 进度统计 | ❌ | ✅ 增强 |
| 阶段过滤 | ❌ | ✅ 增强 |
| 关联文档 | ❌ | ✅ 增强 |
| 时间估算 | ❌ | ✅ 增强 |

## 参数说明

| 参数 | 说明 |
|:---|:---|
| --phase=1 | 只显示 Phase 1 |
| --phase=doing | 只显示进行中 |
| --kanban | Kanban 卡片视图 |
| --progress | 进度统计视图 |
| --simple | 简洁列表 (接近官方) |

## 使用示例

### 查看全部
```bash
vibe-todos
# 显示完整增强视图
```

### 查看进度
```bash
vibe-todos --progress
# 显示进度统计
```

### 按阶段过滤
```bash
vibe-todos --phase=2
# 只显示 Phase 2 的任务
```
