## 审查清单

### 安全: 注入/泄露/权限
### 通用质量
- [ ] 函数<50行, 职责单一?
- [ ] 注释WHY非WHAT?
- [ ] 类型严格, 无any/as any?
- [ ] 错误处理完整, 无空catch?
- [ ] 命名清晰, 表达意图?
- [ ] 无魔术数字, 常量已提取?
- [ ] DRY, 无重复代码?
