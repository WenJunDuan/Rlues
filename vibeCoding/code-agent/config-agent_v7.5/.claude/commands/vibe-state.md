# /vibe-state - 查看状态

查看当前会话状态、工作流进度、TODO状态。

## 触发

```bash
/vibe-state
/vibe-state --full    # 完整状态
/vibe-state --todo    # 只看TODO
/vibe-state --kanban  # 只看看板
```

## 输出格式

### 基本状态

```markdown
## 📊 当前状态

### 会话信息
| 项目 | 值 |
|:---|:---|
| 会话ID | sess_20250112_001 |
| 更新时间 | 10:30:00 |
| 模式 | 🔄 工作流 |

### 工作流
| 项目 | 值 |
|:---|:---|
| 名称 | vibe-code |
| 状态 | 🟢 运行中 |
| 阶段 | Execute (E) |
| 进度 | 40% (2/5) |

### 当前任务
- **ID**: T-003
- **描述**: 实现登录API
- **Owner**: LD
- **进度**: 60%
```

### TODO 状态 (--todo)

```markdown
## 📋 TODO 状态

### 进度
████████░░░░░░░░░░░░ 40% (2/5)

### 列表
- [x] T-001: 数据模型设计 ✅
- [x] T-002: API接口定义 ✅
- [ ] T-003: 实现登录API 🔄 (当前)
- [ ] T-004: 前端页面
- [ ] T-005: 集成测试
```
