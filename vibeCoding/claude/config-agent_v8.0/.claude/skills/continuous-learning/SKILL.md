# Continuous Learning v1

## 触发

每次任务完成 (RIPER R2 阶段) 自动触发。

## 提取内容

1. 错误模式: 什么出了错？怎么修的？
2. 成功模式: 什么方法有效？
3. 用户偏好: 用户修正了什么？

## 写入

提取结果 → experience skill → .knowledge/experience/
