## 🔄 状态流转规则

```
📥 计划中                    🔄 进行中                    ✅ 已完成
┌─────────┐                 ┌─────────┐                 ┌─────────┐
│         │  onTaskStart()  │         │ onTaskComplete()│         │
│  TODO   │ ─────────────▶  │  DOING  │ ─────────────▶  │  DONE   │
│         │                 │         │                 │         │
└─────────┘                 └─────────┘                 └─────────┘
                                 │
                                 │ onTaskPause()
                                 ▼
                            ┌─────────┐
                            │ PAUSED  │
                            └─────────┘
```

### 触发时机

| 事件 | 触发动作 |
|:---|:---|
| 任务开始 | TODO → DOING |
| 任务完成 | DOING → DONE |
| 任务暂停 | DOING → PAUSED |
| 任务恢复 | PAUSED → DOING |
| 任务阻塞 | 添加到阻塞项 |
