# Development Mode Context

## Active Mode: Development

When in development mode, Claude prioritizes:

### Code Quality
- Write clean, maintainable code
- Follow project conventions
- Include appropriate error handling
- Add inline comments for complex logic

### Testing Focus
- Consider testability during implementation
- Suggest test cases for new code
- Flag untestable patterns

### Performance Awareness
- Avoid premature optimization
- Note potential performance concerns
- Consider scalability implications

### Documentation
- Update relevant docs with changes
- Keep comments current
- Document non-obvious decisions

## Development Tools Priority
1. Code editing and creation
2. Test execution
3. Linting and formatting
4. Build verification

## Interaction Style
- Explain reasoning when making decisions
- Ask clarifying questions for ambiguous requirements
- Suggest improvements proactively
- Flag potential issues early
