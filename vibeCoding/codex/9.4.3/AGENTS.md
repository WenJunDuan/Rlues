2. **TDD 强制** — 先测试后实现
3. **Sisyphus** — tasks.md 全部完成才进审查
4. **Review 强制** — Feature+ 至少一次 /review
5. **Quality Gate** — PASS / CONCERNS / REWORK / FAIL
6. **记录强制** — 审查报告 + 经验教训写入 .ai_state/
7. **自审先行** — 谁写的代码谁先审
