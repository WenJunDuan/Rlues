# RIPER Workflow

## Five Phases

| Phase | Focus | Output |
|:---|:---|:---|
| **R**esearch | 理解问题 | 需求分析 |
| **I**nnovate | 探索方案 | 技术选项 |
| **P**lan | 制定计划 | TODO.md |
| **E**xecute | 实施开发 | 代码 |
| **R**eview | 审查验证 | 质量报告 |

## Integration

- Phase transitions trigger cunzhi points
- Each phase uses specific skills
- verification-loop runs at Review phase
